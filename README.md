# WeGrow GraphQL
Implementation of the GraphQL API for WeGrow

## How to Use
```
import gql from 'graphql-tag';
let weGrowGraphQL = require('wegrow_graphql');

const LOGIN = `${gql}${weGrowGraphQL.LOGIN}`;

```
## Implemented Mutations/Queries
- LOGIN
- INDIVIDUAL_FEEDBACK
- SOFT_SKILLS_QUERY
- HARD_SKILLS_QUERY
- RESET_PASSWORD
